Canvas Exercise Filter

Deze extensie voegt filterfunctionaliteit toe aan je Canvas LMS cijferoverzicht.

  • Zoek opdrachten op naam
  • Filter op status (beoordeeld/niet-beoordeeld)
  • Filter op inleverstatus
  • Filter op vak (CSC, PROG, MOD, SmartApp)
  • Filter op inleverdatum
  • Onthoudt je filterinstellingen
Actief! Ga naar je Canvas cijferoverzicht om de filter te gebruiken.
